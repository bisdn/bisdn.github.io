# Installation

For Fedora platforms,

`\`
dnf install python3-sphinx texlive-scheme-full
pip install sphinxcontrib-spelling
\``

# Usage

To make the HTML version,

`\`
make html
\``

To make the PDF version,

`\`
make pdf
\``

To run the spellchecker,

`\`
make spelling
\``
